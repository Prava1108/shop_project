# simple-shop-app

Full-stack example:
- backend: Spring Boot (Gradle) - runs on port 8080
- frontend: React + TypeScript (Vite) - runs on port 5173

## Quick start
1. Start MySQL and create database `shopdb`
2. Edit backend `src/main/resources/application.yml` to set DB credentials and Google OAuth values.
3. Start backend:
   - with Gradle installed: `gradle bootRun`
   - or generate & use Gradle wrapper
4. Start frontend:
   ```
   cd frontend
   npm install
   npm run dev
   ```
